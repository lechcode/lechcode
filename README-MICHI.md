# Lechcode-Website — kurz für Michi 👋

Funktioniert wie bei **wairua**: jede Seite ist **eine HTML-Datei**, kein Build. So bastelst du weiter.

## 👀 Nur ansehen
Datei **`index.html`** doppelklicken → öffnet im Browser. Fertig.
Live ist sie unter **https://lechcode.de**.

## ✏️ Ändern (empfohlen: wie bei wairua)
1. **Repo holen** (einmalig):
   ```
   git clone https://github.com/Lechcode/lechcode.git
   cd lechcode
   ```
2. Ordner in **Claude Code** öffnen, sagen was du willst („ändere den Preis-Text …").
   Claude ändert die Dateien. Die `CLAUDE.md` erklärt Claude den Aufbau automatisch.
3. **Prüfen:** `index.html` im Browser neu laden (Cmd/Strg + Shift + R).
4. **Hochladen / live stellen:**
   ```
   git add -A
   git commit -m "kurz was geändert wurde"
   git push
   ```
   → **GitHub Pages aktualisiert lechcode.de automatisch (~1 Min).**

## 📄 Was auf der Seite steht (Kurz-Version Juni 2026)
- **Oben:** Video-Bühne (Platzhalter — hier kommt unser Video rein)
- **Startangebot:** die ersten 10 Websites für einmalig **500 €**, alles drin
- **Ablauf:** eintragen → Entwurf in 24 h → **dein 30-Min-Gespräch** → live
- **Pflege:** 19 €/Monat, Änderungen per Mail/WhatsApp-Sprachnachricht
- **Formular:** schickt Anfragen an uns (bis der Formular-Key da ist, öffnet es eine fertige E-Mail an info@lechcode.de)

## 🙏 Bitte beachten
- **`noindex` bleibt drin** (Seite ist bei Google unsichtbar, Link funktioniert trotzdem), bis: Video drin + Formular-Key drin + Postfach info@lechcode.de existiert. Dann gemeinsam live gehen.
- **Ton & Look:** warm, „Du", mit Seele · warm-dunkel/„Lagerfeuer" mit Gold/Bernstein — **kein Türkis**.
- Rechtsseiten (Impressum, Datenschutz, AGB) sind verlinkt — Texte bitte nicht löschen.

## 🛟 Wenn was klemmt
- Versehentlich kaputtgemacht? Nichts ist verloren — `git log` zeigt alle Versionen, Claude hilft zurück.
- Im Zweifel einfach **Claude Code fragen** — er kennt die `CLAUDE.md`.
